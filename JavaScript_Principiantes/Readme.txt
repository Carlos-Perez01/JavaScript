Carpeta dedicada a los pilares básicos de Javascript.

Lenguaje: Forma en la que escribes tu codigo.

Framework: Caja de herramientas que te permite montar un proyecto entorno
a una arquitectura definida

Librería: Dependencia externa que permite añadir funcionalidad expecifica en tu
proyecto.

****************************************************************

jQuery: Librería, que sirve para que escribas poco codigo, y permite una mayor compatibilidad,
con menor tamaño de codigo.

Java: Lenguaje Backend que necesita ser compilado.

Vanilla Javascript: Es utilizar Javascript sin nada más.

****************************************************************

Frontend: Parte visible de la página.

Backend: Parte de servidor etc... de la página.

****************************************************************

Server-side: Ejecución en el servidor.(NodeJs)

Client-side: Donde reside el codigo.

****************************************************************

Contenido = HTML

Estilos = CSS

Lógica = JavaScript

****************************************************************
