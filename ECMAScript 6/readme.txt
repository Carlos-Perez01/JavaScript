ECMAScript 6

/*Historia JS, ECMAScript

Qué es ES6, también conocido como ECMAScript 2015, el más reciente estándar del popular lenguaje 
de programación Javascript.

Como ocurre con todos los avances tecnológicos, Javascript ha pasado por diferentes versiones, 
cada una con novedades que mejoraban la anterior. ECMA es quien se encargó de estandarizar Javascript 
y ya va por la versión ES6.

Cabe destacar que la versión anterior, ES5, permaneció durante tanto tiempo que se convirtió en 
la versión más extendida de Javascript en todo tipo de plataformas. Es una versión con un índice 
muy alto de compatibilidad, que la convierte en candidata ideal para ser empleada en la escritura 
de código que valga para todos los tipos de sistemas o navegadores.

Hoy en día, son muchos los sistemas capaces de interpretar la versión que apareció en 2015, la ES6, 
que presenta diversas mejoras respecto al lenguaje y nuevas variantes respecto a la programación. 
Actualmente, ya se está trabajando en la siguiente versión, la ES7, que seguirá mejorando la 
productividad y manejo de Javascript, adaptándose así a las nuevas necesidades.
*/

/*Como trabajar con ES6

A la hora de utilizar ES6 hay que tener en cuenta al usuario final, es decir, 
a aquellas personas que vayan a usar nuestra aplicación.

Con el fin de que el usuario tenga la mejor experiencia usando nuestra web, 
debemos de tener en cuenta el navegador que utiliza. Esto implica que el uso de ciertas 
funcionalidades modernas deben ser compatibles con dicho navegador.

Para ver qué podemos usar y qué no, podemos recurrir a caniuse para visualizar la tabla de 
funcionalidades y soporte en diferentes navegadores junto al marketplace actual.

Otras de las opciones a la hora de trabajar con ES6 podemos también recurrir a Babel para solucionar
la problemática de compatibilidad entre estándares y navegadores. Babel nos permitirá compilar 
nuestro código en ES6 en código compatible para casi todos los navegadores.

https://caniuse.com/ --> Para comprobar la compatibilidad

https://babeljs.io/ --> Para dar compatibilidad mis archivos a los navegadores
*/
